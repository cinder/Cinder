Potential issues

While things do basically appear to be working, there are some things that could potentially break things depending on what you're doing

  1. Including Freetype's ftlcdfil.c file will cause a link error when building Cinder which states `ERROR: Linking globals named 'ft_lcd_padding': symbol multiply defined!`

  2. Shaders are currently not detached after creation due to a possible bug Emscripten that essentially sets the shader handle to null.

  3. While the audio api is technically available, it is still largely un-tested and may still have kinks that need to be worked out.
