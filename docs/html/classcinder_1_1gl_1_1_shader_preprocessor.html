ShaderPreprocessor Class Reference

  • Public Member Functions

Class for parsing and processing GLSL preprocessor directives.

Detected files that need to be included, via the #include directive, are first searched for relative to the directory of the provided source path (if provided), and then relative to the search directories list. By default, the Platform's default assets path is added to this list, and you can add custom search directories with <a href="classcinder_1_1gl_1_1_shader_preprocessor.html#a80f6ed43ef651889bb4f692f5c7a35a3"> addSearchDirectory() </a> .
