Part 3: Advanced Functions

Advanced functions blah blah blah

  • calcBoundingBox(): Calculates the bounding box of the path.
  • calcPreciseBoundingBox(): Calculates the bounding box of the path.
  • contains(): Determines if a given point is within the area of the path.
  • transformed(): Returns a copy of the path with the specified matrix applied.
  • subdivide(): Returns an array of points subdividing the path's curves based on the given tolerance.
